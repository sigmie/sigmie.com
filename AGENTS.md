# Agent Instructions

## Design System

**Always consult [DESIGN.md](./DESIGN.md) before producing or modifying any UI.**

`DESIGN.md` is the single source of truth for visual styling on this site. Every page, layout, and component except the homepage (`resources/js/Pages/Welcome.vue`) must adhere to its tokens, components, and do's/don'ts. When in doubt about a color, type size, spacing value, radius, or shadow, look it up in `DESIGN.md` rather than inventing one.

When adding new UI:
- Use only the colors, type scale, spacing, radii, and shadows defined in `DESIGN.md`.
- Follow the component conventions described there (button shapes, card styles, banners).
- Respect the do's and don'ts section — no new hues, no decorative fonts, no heavy shadows.

When editing existing UI, bring it in line with `DESIGN.md` as part of the change.
